# sinhala-song-lyrics-search-engine

## Data

Each song contains the following data fields.
1. Title
2. Artist
3. Genre
4. Lyrics
5. Music
6. Guitar key (In English)
7. Beat
8. Number of views
9. Number of shares
10. Song lyrics

## Main Usecases
